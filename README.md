# login_kenpo

auto login my health web by headless chrome

## USAGE

setup

```
$ npm install -g @rhythm191/login_kenpo
```

you should set login infomation.

```
$ login_kenpo setting
```

login my health web

```
$ login_kenpo login
```

## Development

test exec

```
$ yarn run exec login
```
